Used to enter the name of a .txt file file containing a prioritized list of cellular (WWAN) and/or Wi-Fi (WLAN) networks, each identified by its associated Access Point Name (APN). The first name on the list is given the highest priority for connection; each subsequent name gets a lesser priority.

Contents of this file replaces any pre-existing values on the device.

Input value rules:

  • String of at least one (1) character containing a list of Access Point Names, one entry per line
