Used to enter the name of the keyboard layout being configured. This is a required field and must contain the name of a layout listed in the "Keyboard layouts" panel on the device. NOTE: Layout names are case sensitive.

To view a list of Keyboard Layouts:

  1. Go to the Android Settings panel.
  2. Select System > Languages & input > Physical keyboard.
  3. Tap the name of the connected external keyboard. The name of the currently selected layout is displayed.
  4. Tap "SET UP KEYBOARD LAYOUTS" to display a list of alternalive layouts (languages).

Parm value input rules:

  • String of at least one alphanumeric character
