MX Feature Reference

Application Management

Manage application white-listing and browser settings.

App Manager

The AppMgr allows you manage the set of user applications that are installed on the device.

4.2 4.4 5.0

Browser Manager

The Browser Manager allows you to configure browser settings like the default home page.

4.4 5.0

Device Configuration

Manage various device configuration settings like power, usb, clock and display behavior.

Analytics

The AnalyticsMgr allows you enable or disable collection of data, in the form of groups of metrics, by the Analytics Engine

4.4 5.0

Audio Manager

The AudioVolUIMgr allows you to add, delete, and replace Audio Profiles and to select the current Audio Profile that will be in effect on the device.

4.4 5.0

Battery Manager

The BatteryMgr allows you to configure the thresholds that will be used to determine when a battery needs to be decommissioned.

4.4 5.0

Clock Manager

The Clock Manager allows you to set the Date, Time, and Time Zone or to configure the device to automatically acquire it via NTP.

4.2 4.4 5.0

Component Manager

The ComponentMgr allows you to configure the state and usage of specific subsystems on the device, such as Ethernet.

4.4 5.0

Display Manager

The DisplayMgr allows you to control the display screen on the device.

4.4 5.0

Power Manager

The PowerMgr allows you to perform power-related actions on the device, such as putting it into Sleep mode.

4.2 4.4 5.0

Touch Manager

The TouchMgr allows you configure the Touch Mode on your device (ex. Finger or Stylus)

4.2 4.4 5.0

MX Core Framework

Access core framework features and how XML settings are managed.

CSP Manager

The CspMgr allows you to query the system to determine the set of CSPs that are present on a given device.

4.2 4.4 5.0

MX Manager

The MX Manager allows you to acquire the version numbers of the MX Management Framework (MXMF) and of the MX CSP itself.

4.2 4.4 5.0

Persistance Manager

The PersistMgr allows you to manage the Request XML Documents that are persistent on a device.

4.2 4.4 5.0

XML Manager

The XmlMgr allows you to specify the Error Handling Mode the MXMS should use when processing a Request XML document.

4.2 4.4 5.0

Security Features

Manage security settings for the device and access to applications.

Access Manager

The AccessMgr enables the configuration of a device to control which user or applications can be used on a given device as well as what those applications can do.

4.2 4.4 5.0

Camera Manager

The CameraMgr allows you to control what cameras, if any, will be allowed to be used.

4.4 5.0

Certificate Manager

The CertMgr allows you to manage certificates and the Android Keystore on a device.

4.2 4.4 5.0

Dev Admin

The DevAdmin allows you to perform certain device administration tasks directly like Screen-Lock timeout interval.

4.4 5.0

Encrypt Manager

The EncryptMgr allows you to manage the Key Storage Database, activate or deactivate Full Storage Card Encryption, and create or delete EFSes.

4.4 5.0

SD Card Manager

The SdCardMgr allows you to control whether one specific External Storage on the device can be used.

4.4 5.0

Threat Manager

The Threat Manager feature allows your application to control what security threats a device actively monitors for and how to respond.

4.4 5.0

USB Manager

The UsbMgr allows you to control which USB functions can be used on the device.

4.4 5.0

UI Settings

Control device settings behavior and remap keys.

KeyMap Manager

The KeyMappingMgr allows you to modify what behavior a given key will exhibit when pressed.

4.4 5.0

Power Key Manager

The PowerKeyMgrallows you to control whether the user will be allowed to use certain menu options that are supported on the Power Key Menu.

4.4 5.0

Settings Manager

The SettingsMgr allows you to control access to items on the System Settings Menu.

4.4 5.0

UI Manager

The UiMgr Feature Type allows you to manage a miscellaneous set of UI configurations, like Clipboard behavior.

4.4 5.0

Wireless

Manage wireless settings like WiFi and GPRS for your device.

Cellular Manager

The CellularMgr allows you to control how a device's Cellular data connection is used.

4.4 5.0

DHCP Manager

The DhcpOptionMgr allows you to configure DHCP Options.

4.4 5.0

GPRS Manager

The GprsMgr allows you to manage APN settings for the devices GPRS network.

4.2 4.4 5.0

WiFi Manager

The Wi-Fi feature type allows you to manage a device's Wi-Fi settings as well as manage the network profiles to be used for connecting and remembering networks.

4.2 4.4 5.0

Wireless Manager

The WirelessMgr allows you to turn various wireless radios On or Off, like Bluetooth, GPRS, NFC, etc.
