GraphicsUtil..::.StoreImage Method (String, String, Int32, Int32)
GraphicsUtil Interface  See Also  Send Feedback
Stores an image from the Windows Mobile device file system to the connected printer as a monochrome image. The image will be stored on the printer at printerDriveAndFileName with the extension GRF. If a drive letter is not supplied, E will be used as the default (e.g. FILE becomes E:FILE.GRF). If an extension is supplied, it is ignored (E:FILE.BMP becomes E:FILE.GRF).
