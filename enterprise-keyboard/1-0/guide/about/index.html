How to Get It

Enterprise Keyboard is preinstalled on Zebra devices running Android 5.0 Lollipop and available as an OS patch for Zebra MC40, TC70 and TC75 devices running Android 4.4 KitKat.

  • For help configuring EKB on a Lollipop device, see Enterprise Keyboard Manager.

  • Download the MC40/TC70/TC75 OS patch at the Enterprise Keyboard download page.

  • For more information, please contact a Zebra representative or partner.
