How To Download

  1. Visit the EKB download page.
  2. Scroll down to the EKB version corresponding to the Android version on the target device.
  3. Download and install the package using the standard procedure.

Remote Management

Enterprise Keyboard can be managed remotely using StageNow and the UI Manager service (requires MX 6.3 or higher) or a company's own enterprise mobile management (EMM) system (if supported by that system).

See the Enterprise Keyboard Manager guide for details.
