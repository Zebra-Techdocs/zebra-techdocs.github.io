Enterprise Keyboard 2.0

支持的设备

EKB 2.0 预安装在运行 Android 8.x Oreo 的以下 Zebra 设备上：

  • MC33
  • PS20
  • TC51
  • TC51 医疗保健
  • TC52
  • TC56
  • TC56 医疗保健
  • TC57
  • TC70x
  • TC72
  • TC75x
  • TC77
  • VC80x

EKB 2.0 预安装在运行 Android 7.x Nougat 的以下 Zebra 设备上：

  • MC33
  • TC51
  • TC51 医疗保健
  • TC56
  • TC56 医疗保健
  • TC70x
  • TC75x
  • VC80x
  • WT6000

注意：对于上面未列出的 Nougat 设备和/或未预安装 EKB 的设备，请参阅下面的“如何下载”。

Nougat 之前的选项

EKB 2.0 作为 LifeGuard 修补程序添加到下面列出的运行 Android 6.x Marshmallow 或较早版本的 Zebra 设备上。

  • TC51
  • TC51 医疗保健
  • TC56
  • TC56 医疗保健
  • TC70x
  • TC75x
  • WT6000

要在上面列出的设备上安装 EKB 2.0：

  1. 访问 LifeGuard 下载页面。
  2. 识别并下载适用于设备及其操作系统的相应修补程序。
  3. 使用下载附带的说明应用修补程序。

对于以下 Nougat 之前的设备，EKB 1.6 作为用户应用程序进行安装：

运行 Android 6.x Marshmouse：

如何下载

  1. 访问 EKB 下载页面。
  2. 向下滚动到与目标设备上的 Android 版本对应的 EKB 版本。
  3. 使用标准过程下载并安装软件包。

远程管理

Enterprise Keyboard 可以通过 EMDK for Android 开发工具以编程方式进行管理，或是使用 StageNow 和 UI Manager 服务（需要 MX 6.3 或更高版本）或公司自己的企业移动管理 (EMM) 系统（如果该系统支持）进行远程管理。

有关详细信息，请参阅 Enterprise Keyboard 管理指南。
