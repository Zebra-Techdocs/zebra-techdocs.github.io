关于 Enterprise Keyboard

Enterprise Keyboard 2.0

概述

Enterprise Keyboard 是一种软输入设备，旨在提供可以将数据输入到 Zebra 设备中的最高效方法。Enterprise Keyboard 在常备 Android 键盘的基础上构建，提供可编程按键，可以快速在语言与按键布局之间切换，并且能够使用设备的任何扫描器将条码数据直接扫描到应用程序中。

Enterprise Keyboard（1.1 及更高版本）可以采用以下方式进行配置：

  • 在设备上手动进行
  • 通过 EMDK for Android 开发工具以编程方式进行
  • 使用 StageNow 和 UI Manager 服务远程进行
  • 通过公司自己的企业移动管理 (EMM) 系统（如果该系统支持）远程进行

Enterprise Keyboard 使用 Android 个人词典进行拼写更正和快捷方式替换，允许使用特定于行业的术语轻松填充词典，以帮助提高键盘输入的速度和准确性。自定义词典可以通过 Zebra 的移动性扩展 (MX) 和 StageNow 工具进行批量填充和大规模部署。

此版本的 Enterprise Keyboard 包括以下语言：

  • 英语（英国）
  • 英语（美国）
  • 法语
  • 德语
  • 意大利语
  • 西班牙语
  • 俄语

了解有关以下各项的更多信息：

  • “可加载”的个人词典

新增的功能

  • Esc 键支持 - 允许将 ESC 键重映射到 ASCII 或 Unicode 字符，或映射到表情符号或是正常工作。关于按键重映射。
  • 多用户模式 - 支持主要用户和辅助用户的部署，后者被阻止在设备上进行 EKB 设置更改。仅在运行 Android 8.x Oreo 的 SDM660 平台设备上受支持。

版本历史记录

v1.9 中的更改

行为更正：
  • 在键入之前导航离开字母键盘之后，自动大写现在可正常工作。
  • 当从字母滑动到符号布局并返回时，键盘弹出窗口不再消失。

v1.8 中新增的功能

注意：Enterprise Keyboard 预安装在运行 Android 7.x Nougat 及更高版本的 Zebra 设备上。对于支持的 Nougat 之前的设备，EKB 作为 LifeGuard 修补程序添加。有关更多信息，请参阅下载页面。

支持的设备

EKB 1.8 增加了对以下设备的支持（并在其上预安装）：

  • WT6000 N

新增的功能

  • 符号键 P1 可以重映射到 ASCII 或 Unicode 字符，或是重映射到表情符号。
  • 每当语音输入键启用时，文本更正建议栏会保持可见，即使未提供建议更正也是如此。
  • 如果禁用语音输入，则仅当键入时才会显示更正建议（如果已启用）。

已停止使用的功能

  • 个性化建议和下一个字词建议功能已删除
    （文本更正建议和其他个人词典功能未受影响）。

v1.6 中新增的功能

“首选项”面板中的新设置：

  • 导航控制在键盘布局之间导航的方式。“选项卡”模式允许通过在布局上滑动（以显示相邻布局）或直接在所需布局选项卡上轻触来切换布局。除了在显示字词更正（如果启用）时，选项卡栏始终可见。“按键”模式允许使用按键来切换布局。此类按键通常在布局的左下角。

  • 选项卡配置使用户或管理员可以启用或禁用数字、字母数字、符号和/或扫描按键布局选项卡，使它们在视图中隐藏以及阻止显示相应布局。

  • 选择首选选项卡会使选定选项卡在每次调用 EKB 时出现。

仅在具有 MX 7.1 及更高版本的设备上才支持导航和选项卡配置功能。

有关支持的设备列表，请参阅 EKB 1.6 下载页面。
